# M-HUZAIFA-TASK-02
While building this project, I found Flexbox easier for arranging the pricing cards in a single row and aligning items using simple properties like justify-content and align-items. It worked well for this one-dimensional layout, but managing equal spacing and responsiveness required a little more effort.
On the other hand, CSS Grid made it much easier to create the three-column pricing section. Using grid-template-columns allowed me to define the layout quickly, and changing it to a single column for mobile screens was simple with media queries. Grid felt more organized and flexible for this type of layout.
Overall, both techniques were useful, but for this pricing page, CSS Grid was more efficient for the main layout, while Flexbox was more suitable for smaller components like the navigation bar and aligning content inside the cards.
